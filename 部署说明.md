## 🎯 快速开始

### 方式 A：一键部署（推荐）

```bash
# 1. 进入项目目录
cd /home/node/.openclaw/workspace/ai-side-laravel-max

# 2. 执行部署脚本
bash deploy.sh

# 3. 按提示输入 MySQL 密码
# 4. 等待部署完成
```

### 方式 B：分步部署

```bash
# 1. 创建数据库
mysql -u root -p -e "CREATE DATABASE \`ai-side-laravel-max\` DEFAULT CHARACTER SET utf8mb4;"

# 2. 创建容器
docker run -d \
  --name ai-side-laravel-max \
  -p 8082:80 \
  -v /home/node/.openclaw/workspace/ai-side-laravel-max:/var/www/html \
  -e APACHE_DOCUMENTROOT=/var/www/html/public \
  laravelsail/php82-composer:latest

# 3. 安装依赖
docker exec -it ai-side-laravel-max bash -c "composer install --no-dev --optimize-autoloader"

# 4. 生成密钥
docker exec -it ai-side-laravel-max bash -c "php artisan key:generate"

# 5. 配置 .env
# 编辑 /home/node/.openclaw/workspace/ai-side-laravel-max/.env

# 6. 运行迁移
docker exec -it ai-side-laravel-max bash -c "php artisan migrate --force"

# 7. 设置权限
docker exec -it ai-side-laravel-max bash -c "chmod -R 775 storage/ bootstrap/cache/"

# 8. 优化缓存
docker exec -it ai-side-laravel-max bash -c "php artisan config:cache && php artisan route:cache && php artisan view:cache"
```

