## 🎯 设计规范

### 1. 卡片样式

**标准：**
```html
<div class="bg-white rounded-2xl shadow-xl p-6 border border-gray-100 card-hover">
```

**应用：**
- ✅ 所有卡片使用 rounded-2xl（16px 圆角）
- ✅ 所有卡片使用 shadow-xl（大阴影）
- ✅ 所有卡片使用 border-gray-100（细边框）
- ✅ 所有卡片使用 card-hover（悬停效果）

**效果：**
```
默认：shadow-xl + border-gray-100
悬停：shadow-2xl + transform -translate-y
```

---

### 2. 按钮样式

**主按钮（CTA）：**
```html
<button class="gradient-bg text-white px-8 py-4 rounded-full font-bold hover:opacity-90 transition shadow-xl transform hover:scale-105">
```

**次按钮：**
```html
<button class="border-2 border-primary-600 text-primary-600 px-8 py-4 rounded-full font-bold hover:bg-primary-50 transition">
```

**应用：**
- ✅ 所有 CTA 按钮使用 gradient-bg
- ✅ 所有按钮使用 rounded-full（圆形）
- ✅ 所有按钮使用 font-bold（粗体）
- ✅ 所有按钮使用 shadow-xl（阴影）
- ✅ 所有按钮使用 hover:scale-105（悬停放大）

- ✅ 所有菜单项使用 hover 渐变背景

## ✨ 交互效果

### 1. 卡片悬停

```css
.card-hover {
    transition: all 0.3s cubic-bezier(0.4, 0, 0.2, 1);
}

.card-hover:hover {
    transform: translateY(-4px);
    box-shadow: 0 20px 25px -5px rgba(0, 0, 0, 0.1);
}
```

**应用：**
- ✅ 所有卡片使用 card-hover
- ✅ 悬停时上移 4px
- ✅ 悬停时阴影加深

---

### 2. 按钮悬停

```css
button {
    transition: all 0.3s ease;
}

button:hover {
    transform: scale(1.05);
    opacity: 0.9;
}
```

**应用：**
- ✅ 所有按钮使用 hover:scale-105
- ✅ 渐变按钮使用 hover:opacity-90

---

### 3. 菜单悬停

```css
a {
    transition: all 0.3s ease;
}

a:hover {
    background: linear-gradient(to right, #f5f3ff, #fce7f3);
}
```

**应用：**
- ✅ 所有菜单项使用 hover 渐变背景

## 📱 响应式设计

### 断点

```css
sm:  640px   (手机横屏)
md:  768px   (平板)
lg:  1024px  (桌面)
xl:  1280px  (大桌面)
2xl: 1536px  (超大桌面)
```

### 布局

```html
<!-- 移动端：单列 -->
<div class="grid grid-cols-1">

<!-- 平板：双列 -->
<div class="md:grid-cols-2">

<!-- 桌面：三列 -->
<div class="lg:grid-cols-3">
```

**应用：**
- ✅ 所有网格使用响应式布局
- ✅ 所有卡片自适应宽度
- ✅ 所有文字大小适中
