## 十、快速启动

### 10.1 环境准备

```bash
# 1. 创建数据库
mysql -u root -p -e "CREATE DATABASE \`ai-side-laravel-max\`;"

# 2. 进入项目目录
cd /home/node/.openclaw/workspace/ai-side-laravel-max

# 3. 安装依赖
composer install --no-dev --optimize-autoloader

# 4. 生成密钥
php artisan key:generate

# 5. 配置 .env
# 编辑数据库连接、邮件配置等

# 6. 运行迁移
php artisan migrate
```

### 10.2 测试验证

```bash
# 启动开发服务器
php artisan serve --host=0.0.0.0 --port=8082
```
