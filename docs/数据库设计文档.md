## 二、表结构详细

### 2.1 用户相关表

#### 2.1.1 users - 用户表

**来源：** ✅ 1.0 复用（需新增字段）

```sql
CREATE TABLE users (
    id BIGINT UNSIGNED AUTO_INCREMENT PRIMARY KEY,
    name VARCHAR(255) NOT NULL COMMENT '用户昵称',
    email VARCHAR(255) UNIQUE COMMENT '邮箱',
    phone VARCHAR(20) UNIQUE COMMENT '手机号',
    email_verified_at TIMESTAMP NULL COMMENT '邮箱验证时间',
    phone_verified_at TIMESTAMP NULL COMMENT '手机验证时间',
    password VARCHAR(255) COMMENT '密码（加密）',
    role VARCHAR(50) DEFAULT 'user' COMMENT '角色：admin/user/vip',
    avatar VARCHAR(500) COMMENT '头像 URL',
    bio TEXT COMMENT '个人简介',
    enterprise_wechat_userid VARCHAR(100) COMMENT '企业微信用户 ID',
    theme_preference VARCHAR(50) DEFAULT 'default' COMMENT '主题偏好',
    remember_token VARCHAR(100) COMMENT '记住登录 token',
    created_at TIMESTAMP NULL,
    updated_at TIMESTAMP NULL,
    
    INDEX idx_email (email),
    INDEX idx_phone (phone),
    INDEX idx_role (role)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci;
```

**字段说明：**

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| id | BIGINT | ✅ | 主键 |
| name | VARCHAR(255) | ✅ | 用户昵称 |
| email | VARCHAR(255) | ❌ | 邮箱（登录用） |
| phone | VARCHAR(20) | ❌ | 手机号 |
| role | VARCHAR(50) | ✅ | 角色：admin/user/vip/svip |
| enterprise_wechat_userid | VARCHAR(100) | ❌ | 企业微信绑定 ID |
| theme_preference | VARCHAR(50) | ✅ | 主题偏好：default/blue/green/orange/dark |
