## 八、常见问题

### 8.1 迁移失败

**问题：** `SQLSTATE[42S01]: Base table or view already exists`

**解决：**
```bash
# 删除所有表后重新迁移
mysql -u root -p ai-side-laravel-max -e "DROP DATABASE \`ai-side-laravel-max\`; CREATE DATABASE \`ai-side-laravel-max\` DEFAULT CHARACTER SET utf8mb4;"
php artisan migrate --force
```

### 8.2 外键约束错误

**问题：** `Cannot add foreign key constraint`

**解决：**
- 检查表创建顺序
- 确保被引用的表已创建
- 检查字段类型是否匹配

### 8.3 字符集不匹配

**问题：** `Specified key was too long`

**解决：**
```bash
# 确保使用 utf8mb4
mysql -u root -p -e "ALTER DATABASE \`ai-side-laravel-max\` CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;"
```
