#### 2.3.2 发布流程

```
┌─────────────────────────────────────────┐
│  1. 点击"发布"按钮                      │
│     ↓                                   │
│  2. 选择发布类型                        │
│     ↓                                   │
│  3. 填写内容                            │
│     - 标题（必填）                      │
│     - 分类（必填）                      │
│     - 内容（必填，支持 Markdown）       │
│     - 标签（可选）                      │
│     - 封面图（可选）                    │
│     - 附件（可选）                      │
│     ↓                                   │
│  4. 设置可见性                          │
│     - 公开（所有人可见）                │
│     - VIP 专属（仅 VIP 可见）             │
│     - 私密（仅自己可见）                │
│     ↓                                   │
│  5. 提交审核                            │
│     - 显示审核提示                      │
│     - 预计审核时间                      │
│     ↓                                   │
│  6. 等待审核                            │
│     - 可在"我的发布"查看状态            │
│     - 审核通过 → 前台展示               │
│     - 审核不通过 → 站内信通知原因       │
│                                         │
└─────────────────────────────────────────┘

审核时效：
- 工作日：24 小时内
- 周末/节假日：48 小时内
```
