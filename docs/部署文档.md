## 二、快速启动命令

### 2.1 首次部署

```bash
# 1. 进入项目目录
cd /home/node/.openclaw/workspace/ai-side-laravel-max

# 4. 配置 .env 文件

# 9. 启动开发服务器（测试用）
php artisan serve --host=0.0.0.0 --port=8082
```

## 四、Nginx 配置

### 4.1 Nginx 配置示例

```nginx
server {
    listen 80;
    server_name aifyqbj.calmpu.com;
    root /home/node/.openclaw/workspace/ai-side-laravel-max/public;
    index index.php index.html;

    # MAX 版本路径
    location /max {
        try_files $uri $uri/ /index.php?$query_string;
    }

    # API 路由
    location /api {
        try_files $uri $uri/ /index.php?$query_string;
    }

    # 禁止访问隐藏文件
    location ~ /\. {
        deny all;
    }

    # PHP 处理
    location ~ \.php$ {
        fastcgi_pass unix:/var/run/php/php8.2-fpm.sock;
        fastcgi_index index.php;
        include fastcgi_params;
        fastcgi_param SCRIPT_FILENAME $document_root$fastcgi_script_name;
        fastcgi_param PATH_INFO $fastcgi_path_info;
        fastcgi_param PATH_TRANSLATED $document_root$fastcgi_path_info;
    }

    # 静态资源缓存
    location ~* \.(jpg|jpeg|png|gif|ico|css|js|svg|woff|woff2|ttf|eot)$ {
        expires 1y;
        add_header Cache-Control "public, immutable";
    }

    # 日志
    access_log /var/log/nginx/aifyqbj-access.log;
    error_log /var/log/nginx/aifyqbj-error.log;
}
```

## 五、定时任务配置

### 5.1 Laravel 调度器

**编辑 `app/Console/Kernel.php`：**

```php
protected function schedule(Schedule $schedule): void
{
    // AI 文章采集（每 30 分钟）
    $schedule->command('ai:fetch-articles')
             ->everyThirtyMinutes()
             ->timezone('Asia/Shanghai');
    
    // GitHub 项目采集（每 30 分钟）
    $schedule->command('ai:fetch-projects')
             ->everyThirtyMinutes()
             ->timezone('Asia/Shanghai');
    
    // 每日 AI 日报（每天 09:00）
    $schedule->command('wechat:send-daily-digest')
             ->dailyAt('09:00')
             ->timezone('Asia/Shanghai');
    
    // 清理旧数据（每天 03:00）
    $schedule->command('cleanup:old-data')
             ->dailyAt('03:00')
             ->timezone('Asia/Shanghai');
}
```

### 5.2 系统 Cron 配置

```bash
# 编辑 crontab
crontab -e

# 添加 Laravel 调度器（每分钟执行一次）
* * * * * cd /home/node/.openclaw/workspace/ai-side-laravel-max && php artisan schedule:run >> /dev/null 2>&1

# 或者直接配置具体任务
*/30 * * * * cd /home/node/.openclaw/workspace/ai-side-laravel-max && php artisan ai:fetch-articles >> /dev/null 2>&1
*/30 * * * * cd /home/node/.openclaw/workspace/ai-side-laravel-max && php artisan ai:fetch-projects >> /dev/null 2>&1
```

## 七、备份策略

### 7.1 数据库备份

```bash
#!/bin/bash
# backup-db.sh

BACKUP_DIR="/backup/ai-side-laravel-max"
DATE=$(date +%Y%m%d-%H%M%S)
DB_NAME="ai-side-laravel-max"
DB_USER="root"
DB_PASS="your_password"

# 创建备份目录
mkdir -p $BACKUP_DIR

# 备份数据库
mysqldump -u $DB_USER -p$DB_PASS $DB_NAME > $BACKUP_DIR/db-$DATE.sql

# 压缩备份
gzip $BACKUP_DIR/db-$DATE.sql

# 删除 30 天前的备份
find $BACKUP_DIR -name "db-*.sql.gz" -mtime +30 -delete

echo "Database backup completed: db-$DATE.sql.gz"
```

### 7.2 文件备份

```bash
#!/bin/bash

PROJECT_DIR="/home/node/.openclaw/workspace/ai-side-laravel-max"
```
