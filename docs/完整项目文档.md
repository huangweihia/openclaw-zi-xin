## 二、技术架构

### 2.1 系统架构

```
┌─────────────────────────────────────────┐
│  OpenClaw 自动化采集                     │
│  （定时任务）                           │
│                                         │
│  ↓ Webhook 推送                         │
│                                         │
│  Laravel 后端                            │
│  ├── 内容接收                           │
│  ├── 数据存储                           │
│  ├── 用户管理                           │
│  └── 推送调度                           │
│                                         │
│  ↓ 推送                                 │
│                                         │
│  邮件（SMTP）                           │
│  企业微信（Webhook）                    │
│                                         │
│  ↓ 用户                                 │
│                                         │
│  网站访问 → 注册 → 付费 → 留存          │
└─────────────────────────────────────────┘
```

### 2.2 技术栈

| 前端 | Blade + Tailwind | Laravel 内置 |

| 数据库 | MySQL 8 | Docker |
| 采集 | OpenClaw | 自动化 |
| 邮件 | SMTP（QQ 邮箱） | 免费 |
| 推送 | 企业微信 | 免费 |
| 部署 | Docker | 已有服务器 |

## 三、数据库设计

### 3.1 核心表

#### users（用户表）
```sql
CREATE TABLE users (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    name VARCHAR(255) NOT NULL,
    email VARCHAR(255) UNIQUE,
    phone VARCHAR(20) UNIQUE,
    password VARCHAR(255),
    role VARCHAR(50) DEFAULT 'user',
    email_verified_at TIMESTAMP,
    phone_verified_at TIMESTAMP,
    enterprise_wechat_userid VARCHAR(100),
    last_login_at TIMESTAMP,
    created_at TIMESTAMP,
    updated_at TIMESTAMP
);
```

#### side_hustle_cases（副业案例）
```sql
CREATE TABLE side_hustle_cases (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    title VARCHAR(255) NOT NULL,
    summary TEXT NOT NULL,
    content LONGTEXT NOT NULL,
    category VARCHAR(50) DEFAULT 'online',
    difficulty VARCHAR(20) DEFAULT 'medium',
    time_commitment VARCHAR(100),
    startup_cost VARCHAR(100),
    revenue_model VARCHAR(255),
    estimated_monthly_income INT,
    actual_income INT,
    income_screenshots JSON,
    steps JSON,
    tools_needed JSON,
    common_pitfalls JSON,
    is_verified BOOLEAN DEFAULT FALSE,
    is_vip_only BOOLEAN DEFAULT TRUE,
    view_count INT DEFAULT 0,
    like_count INT DEFAULT 0,
    save_count INT DEFAULT 0,
    created_at TIMESTAMP,
    updated_at TIMESTAMP,
    INDEX idx_category_vip (category, is_vip_only),
    INDEX idx_verified (is_verified)
);
```

#### ai_tool_monetization（工具变现）
```sql
CREATE TABLE ai_tool_monetization (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    tool_name VARCHAR(255) NOT NULL,
    tool_url VARCHAR(500) NOT NULL,
    tool_logo VARCHAR(500),
    category VARCHAR(50) DEFAULT 'text',
    description TEXT NOT NULL,
    monetization_scenarios JSON NOT NULL,
    prompt_templates JSON,
    delivery_standards JSON,
    pricing_guide JSON,
    client_channels JSON,
    is_domestic BOOLEAN DEFAULT TRUE,
    pricing_model VARCHAR(50),
    popularity_score INT DEFAULT 50,
    is_vip_only BOOLEAN DEFAULT FALSE,
    view_count INT DEFAULT 0,
    created_at TIMESTAMP,
    updated_at TIMESTAMP,
    INDEX idx_category_domestic (category, is_domestic),
    INDEX idx_popularity (popularity_score)
);
```

#### private_traffic_sops（运营 SOP）
```sql
CREATE TABLE private_traffic_sops (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    title VARCHAR(255) NOT NULL,
    platform VARCHAR(50) DEFAULT 'wechat',
    type VARCHAR(50) DEFAULT 'growth',
    summary TEXT NOT NULL,
    content LONGTEXT NOT NULL,
    checklist JSON,
    templates JSON,
    tools_recommended JSON,
    metrics JSON,
    case_studies JSON,
    difficulty_level INT DEFAULT 3,
    estimated_time INT,
    is_vip_only BOOLEAN DEFAULT TRUE,
    view_count INT DEFAULT 0,
    save_count INT DEFAULT 0,
    created_at TIMESTAMP,
    updated_at TIMESTAMP,
    INDEX idx_platform_type (platform, type),
    INDEX idx_difficulty (difficulty_level)
);
```

#### premium_resources（付费资源）
```sql
CREATE TABLE premium_resources (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    title VARCHAR(255) NOT NULL,
    type VARCHAR(50) DEFAULT 'course_notes',
    source VARCHAR(255),
    description TEXT NOT NULL,
    content LONGTEXT NOT NULL,
    files JSON,
    tags JSON,
    original_price VARCHAR(50),
    is_summarized BOOLEAN DEFAULT TRUE,
    curator_note TEXT,
    quality_score INT DEFAULT 5,
    is_vip_only BOOLEAN DEFAULT TRUE,
    download_count INT DEFAULT 0,
    view_count INT DEFAULT 0,
    created_at TIMESTAMP,
    updated_at TIMESTAMP,
    INDEX idx_type_vip (type, is_vip_only),
    INDEX idx_quality (quality_score)
);
```

#### subscriptions（订阅表）
```sql
CREATE TABLE subscriptions (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    user_id BIGINT NOT NULL,
    plan VARCHAR(50) DEFAULT 'free',
    starts_at TIMESTAMP,
    ends_at TIMESTAMP,
    payment_method VARCHAR(50),
    transaction_id VARCHAR(255),
    amount INT DEFAULT 0,
    status VARCHAR(50) DEFAULT 'active',
    created_at TIMESTAMP,
    updated_at TIMESTAMP,
    FOREIGN KEY (user_id) REFERENCES users(id),
    INDEX idx_user_plan (user_id, plan),
    INDEX idx_status (status)
);
```
